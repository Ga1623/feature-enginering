# Titanic Survival Prediction - Feature Engineering Project

## Objective
To clean, engineer, and select features from Titanic dataset to prepare for machine learning modeling.

## Steps Followed
- Handled missing values using median/mode and unknown labels
- Removed unnecessary columns
- Engineered new features:
  - FamilySize
  - IsAlone
  - Title
  - Deck
  - AgeGroup
  - FarePerPerson
- Encoded categorical variables using one-hot encoding
- Scaled and transformed skewed features
- Performed feature selection using correlation and Random Forest
